# STM32-GPIO-Digital-Output
📌 Project Overview
This experiment demonstrates the basic configuration of a GPIO pin as digital output on an STM32 microcontroller and implementation of LED blinking.
It illustrates the fundamental concept of embedded systems where a microcontroller interacts with external hardware through GPIO pins.
This is the first step toward understanding real-time embedded hardware control using STM32.

🎯 Aim-  To configure a GPIO pin as digital output and implement LED blinking using STM32 microcontroller to understand basic embedded system operations. 

🧰 Tools & Software Used- STM32 Discovery Board, STM32CubeIDE, STM32CubeMX, Embedded C (HAL libraries)
